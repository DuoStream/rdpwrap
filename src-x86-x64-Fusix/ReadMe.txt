======================================================================== 动态链接库概述 RDPWrap 项目
这个 DLL 库 RDPWrap 是使用应用程序向导自动创建的。

在这个文件中，展示了 RDPWrap 应用程序的所有文件内容的概述。

RDPWrap.vcxproj 这是 VC++ 项目的主要文件，由应用程序向导创建。它包含了使用的 Visual C++ 语言版本信息，以及使用应用程序向导选择的平台、配置和项目功能信息。

RDPWrap.vcxproj.filters 这是 VC++ 项目的过滤器文件，由应用程序向导创建。它包含了项目文件和过滤器之间的映射关系。这些映射关系用于 IDE 环境中将相同扩展名的文件分组到同一个节点中（例如 CPP 文件映射到“源文件”过滤器）。

RDPWrap.cpp 这是 DLL 库的主要源文件。

在创建这个 DLL 库时，不执行符号导出。因此，在构建时不创建 LIB 文件。如果在这个项目中需要定义对另一个项目的依赖项，需要添加符号导出代码以确保创建导出库，或者在项目的“编译器”页的“属性页”对话框中将“跳过输入库”属性设置为“是”。

///////////////////////////////////////////////////////////////////////////// 其他标准文件：

StdAfx.h, StdAfx.cpp 这些文件用于构建预编译头文件（PCH）RDPWrap.pch 和预编译类型文件 StdAfx.obj。
